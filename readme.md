# Canvasly Server 🎨🚀

This is the backend repository for **Canvasly**, a specialized platform for artists to showcase their artworks, manage favorites, and follow other artists. It is built using **Node.js**, **Express.js**, and **MongoDB**.

## 🔗 Links
- **Live Server URL:** https://canvasly-66acd.web.app/
- **Client Repository:** https://github.com/arafbinmasud/canvasly-client

---

## 🛠️ Technologies Used
- **Node.js**: Runtime environment.
- **Express.js**: Web framework for building APIs.
- **MongoDB**: NoSQL database for data storage.
- **Firebase Admin SDK**: For secure user authentication and token verification.
- **CORS & Dotenv**: For handling cross-origin requests and environment variables.

---

## 🌟 Key Features of the API
- **JWT/Firebase Token Verification**: Secure private routes using custom middleware.
- **Artwork Management**: Full CRUD operations (Create, Read, Update, Delete) for artworks.
- **User Engagement**: Endpoints for handling Likes, Favorites, and Follow/Unfollow logic.
- **Optimized Data Retrieval**: Specialized routes for featured artworks, artist profiles, and follower lists.
- **Search & Filter**: Backend support for searching artworks by title or artist and filtering by category.
